# **App Name**: Family Finance Tracker

## Core Features:

- Daily Income Recording: Record daily income with date and amount, with automatic calculation of profit after deducting fuel expenses.
- Fuel Expense Tracking: Track fuel expenses by date and amount, deducting from daily income.
- Household Expense Management: Add and categorize household expenses (Food, Electricity, Rent, etc.) with optional notes.
- EMI Management: Manage multiple EMIs (e.g., Scooty Finance, Van Finance) with details like EMI Name, Vehicle Type, Monthly EMI Amount, Total Months, and Start Date.
- Automated EMI Deduction: Automatically deduct monthly EMI payments from the total balance, tracking paid and remaining months/amount.
- Financial Dashboard: Display key metrics like Total Income, Total Fuel Expense, Total Home Expense, Total EMI Paid, and Net Balance/Savings with monthly summaries and charts.
- Data Storage: Use Firestore to store income, fuel expenses, home expenses, and EMI data.

## Style Guidelines:

- Primary color: Light blue (#A7D1FF) for a calm, trustworthy feel.
- Background color: White (#FFFFFF) for a clean, minimal look.
- Accent color: Soft grey (#808080) for borders and subtle highlights.
- Body and headline font: 'PT Sans', a humanist sans-serif providing modern readability and a little warmth.
- Clean cards and tables to present data clearly and organized.
- Responsive design to ensure usability on both mobile and desktop devices.
